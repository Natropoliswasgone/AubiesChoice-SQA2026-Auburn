# CFR_TEST – Verification & Validation Project

**21 CFR Atomic Rules | SQA 2026 – Auburn University**

## Project Overview

This project focuses on **Verification & Validation (V&V)** of regulatory requirements from the Code of Federal Regulations (CFR). We extract structured requirements from a markdown file, generate atomic rules, create test cases, and validate correctness using automated scripts and GitHub Actions.
